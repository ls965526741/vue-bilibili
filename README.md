# 仿 B 站项目

### 使用技术

```js
vue
axios
mockjs
```

## 项目依赖

```
npm install
```

### 运行项目

```
npm run serve
```

### 实现功能

#### 组件

- 自己封装组件为主，结合 vant 实现所有功能
- 组件按需加载

#### 数据

- 使用 mock 结合 axios 模拟前后端交互

#### 评论

- 实现评论的展示
- 发起评论，用户匿名评论等功能

#### 个人中心

- 实现换肤静态页面的编写
- 实现账号密码，手机号验证码登陆页面的编写
- 实现个人主页的编写
